# SWCNT Lite 安装指南

## Web 版

打开：

- https://hli2022.github.io/swcnt-mobile-lite/

## Android 安装

下载 APK：

- https://github.com/HLi2022/swcnt-mobile-lite/releases/download/v1.0.0/app-debug.apk

下载后在安卓手机点击安装即可。若提示“未知来源”，在系统设置里允许本次安装。

## iOS 安装（推荐）

1. 用 Safari 打开 Web 版链接
2. 点击“分享”
3. 选择“添加到主屏幕”

## iOS 原生安装（可选）

1. 在 Mac 上打开 Xcode
2. 打开 `ios/App/App.xcworkspace`
3. 连接 iPhone 后运行安装
