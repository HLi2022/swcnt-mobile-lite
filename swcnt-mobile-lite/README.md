# SWCNT Lite Installation Guide

## Web

Open:

- https://hli2022.github.io/swcnt-mobile-lite/

## Android

Download APK:

- https://github.com/HLi2022/swcnt-mobile-lite/releases/download/v1.0.0/app-debug.apk

After download, install on your Android phone. If prompted, allow this install from unknown sources.

## iOS (Recommended)

1. Open the web link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

## iOS Native Install (Optional)

1. Open Xcode on Mac.
2. Open `ios/App/App.xcworkspace`.
3. Connect iPhone and run from Xcode.
