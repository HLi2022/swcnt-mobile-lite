# SWCNT Lite Installation Guide

## About

SWCNT Lite is a lightweight app for quick single-walled carbon nanotube (SWCNT) checks.  
It lets you input chirality `(n,m)`, view fitted pseudo-Voigt absorption spectra, overlay multiple chiralities, and read core tube properties like diameter, chiral angle, RBM, and optical transitions.

## Web

Open:

- https://hli2022.github.io/swcnt-mobile-lite/

## Android

Download APK:

- https://github.com/HLi2022/swcnt-mobile-lite/releases/download/v1.0.0/app-debug.apk

After download, install on your Android phone. If prompted, allow this install from unknown sources.

## iOS (Recommended)

1. Open the web link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

## iOS Native Install (Optional)

1. Open Xcode on Mac.
2. Open `ios/App/App.xcworkspace`.
3. Connect iPhone and run from Xcode.
