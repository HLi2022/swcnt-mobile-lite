# SWCNT Lite Installation Guide

## Web

Open directly:

- https://hli2022.github.io/swcnt-mobile-lite/

## Android

Download APK:

- https://github.com/HLi2022/swcnt-mobile-lite/releases/download/v1.0.0/app-debug.apk

If Android shows an "unknown source" warning, allow this install in system settings.

## iOS (No Developer Account Needed)

1. Open the web link in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

This works like a local app icon on your iPhone/iPad.

## iOS Native Install (Optional)

1. Open Xcode on Mac.
2. Open `swcnt-mobile-lite/ios/App/App.xcworkspace`.
3. Connect iPhone and run from Xcode.
